# AEROTICA Documentation

Welcome to the AEROTICA documentation! AEROTICA is an intelligent computational framework for atmospheric kinetic energy mapping and aero-elastic resilience.

## Overview

AEROTICA integrates nine analytically independent parameters into a single Atmospheric Kinetic Efficiency (AKE) index, powered by a Physics-Informed Neural Network (PINN) architecture.

## Key Features

- **9-parameter AKE index** with Bayesian-optimized weights
- **Physics-Informed Neural Networks** for real-time wind field prediction
- **Gust pre-alerting** with 4-6 minute lead time
- **Urban wind assessment** using high-resolution LiDAR
- **Offshore wind farm optimization** with 34% wake model improvement
